# Electric Vehicles Market Overview Dashboard - 2025

## Project Overview
This project is an interactive Excel dashboard analyzing the Electric Vehicle market in 2025.

The dashboard provides insights into:
- EV distribution by car body type
- Average range by brand
- Top 10 longest-range EVs
- Average fast charging power by segment
- Key KPI metrics

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Key Insights
- SUVs dominate the EV market
- Premium brands provide higher average driving range
- Luxury segments support higher fast-charging power

## Dashboard Features
- Interactive slicers
- Dynamic filtering
- KPI visualization
- Navigation buttons

## Dataset
Public EV dataset used for educational and analytical purposes.
